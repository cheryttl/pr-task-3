# Smart Number Transformation Dashboard (Project 3)

Современная панель управления на базе **React 19**, демонстрирующая работу со сложными компонентами интерфейса, библиотекой **shadcn/ui** и динамическим изменением стилей.

## 🌟 Основные особенности
* **Множественные виджеты:** Приложение рендерит список независимых компонентов `NumberWidget` с разными начальными значениями.
* **Интеллектуальная стилизация:** Цвет кнопок динамически меняется в зависимости от введенного значения:
  * `< 0`: Критичный уровень (**Красный**)
  * `< 50`: Средний уровень (**Серый**)
  * `< 100`: Повышенный уровень (**Темный нейл**)
  * `100+`: Высокий уровень (**Янтарный**)
* **Математическая обработка:** Автоматический расчет по формуле: $y = x \cdot 2.5 + 10$.
* **Glassmorphism UI:** Использование эффекта `backdrop-blur`, мягких теней и прозрачности для создания современного "стеклянного" интерфейса.
* **Адаптивная темная тема:** Полная поддержка `dark mode` через Tailwind CSS.

## 🛠 Технологический стек
* **UI Библиотеки:** [shadcn/ui](https://ui.shadcn.com/) (на базе Radix UI) для профессиональных компонентов.
* **React 19** + **TypeScript** — строгая типизация и высокая производительность.
* **Tailwind CSS v4** — современная утилитарная стилизация.
* **Lucide React** — набор контурных иконок.
* **CVA (class-variance-authority)** — управление сложными состояниями классов.
